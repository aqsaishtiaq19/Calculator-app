# 🧮 Calculator App

A simple, responsive calculator built with **HTML, CSS, and JavaScript**. It performs basic arithmetic operations through a clean, user-friendly interface.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

## ✨ Features

- ➕➖✖️➗ Basic operations: addition, subtraction, multiplication, division
- ⚡ Instant calculation
- 📱 Responsive design — works across screen sizes
- 🎨 Simple, clean UI

## 🛠️ Tech Stack

- **HTML** — structure and layout
- **CSS** — styling and responsive design
- **JavaScript** — calculation logic and DOM interaction

## 🚀 How to Use

1. Clone the repository
   ```bash
   git clone https://github.com/aqsaishtiaq19/calculator-app.git
   ```
2. Open `index.html` in your browser — no installation or dependencies required.
3. Start calculating.
